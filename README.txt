== Description ==

Premium multi-brand café WordPress theme with homepage sections, page templates, and a companion slider plugin for Featured Treats.

== Installation ==

1. Upload `tutti-frutti-theme` to `/wp-content/themes/`
2. Upload `tutti-frutti-slider` to `/wp-content/plugins/`
3. In WordPress Admin: **Plugins → Activate "Tutti Frutti Slider"**
4. **Appearance → Themes → Activate "Tutti Frutti Cafe"**
5. On activation, demo pages and primary menu are created automatically
6. Verify **Settings → Reading**: "Your homepage displays" = A static page, Homepage = **Home**
7. Visit the site (internet required for Unsplash demo images)

== Homepage Sections ==

- Hero with CTAs
- Explore Our Brands (4 cards)
- A Café for Every Moment
- Featured Treats slider (plugin shortcode)
- Late Nights / High Tea promo banners

== Page Templates ==

- About
- Brands
- Pio Coffee Brand
- Order Online
- Rewards
- Careers
- Franchise
- Contact

Assign templates under **Pages → Edit → Page Attributes → Template**.

== Slider Plugin ==

Shortcode: `[tutti_slider group="featured-treats" slides="10"]`

Manage slides: **WP Admin → TF Slides**

Demo slides are created on first plugin activation (5 treats).

== Demo Images ==

All demo images use online Unsplash URLs defined in `inc/demo-images.php`. Change URLs there to swap placeholders site-wide.

== Customizer ==

Appearance → Customize:
- Site Logo
- Contact Phone / Email
- Footer Copyright Text
